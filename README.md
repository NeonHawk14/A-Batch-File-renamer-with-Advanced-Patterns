 Advanced Batch File Renamer

A React-based file renaming tool with regex support, sequential numbering, and case conversion.

## Features
- Multiple renaming rules
- Regular expression support
- Counter sequences with padding
- Case conversion (upper, lower, title, camel)
- Live preview
- Export to bash/batch scripts

## Usage
1. Upload files
2. Add renaming rules
3. Configure counter settings
4. Generate preview
5. Download execution scripts
